Documentation de AEGIS

Explorez toutes les commandes pour gérer et modérer votre serveur Discord avec un préfixe personnalisé.

Bienvenue dans la Documentation de AEGIS

AEGIS est un bot de modération puissant et facile à utiliser, conçu pour aider les administrateurs de serveurs Discord à gérer et modérer leurs communautés. Découvrez ci-dessous toutes les commandes disponibles pour maintenir l'ordre sur votre serveur.

Installation

Pour ajouter AEGIS à votre serveur Discord, suivez les étapes suivantes :

  • 1. Cliquez sur le bouton ci-dessous pour accéder à la page d'ajout du bot.
  • 2. Connectez-vous à votre compte Discord si ce n'est pas déjà fait.
  • 3. Sélectionnez le serveur sur lequel vous souhaitez ajouter AEGIS.
  • 4. Cliquez sur "Autoriser" pour accorder les permissions nécessaires.

Commandes avec Préfixe

Voici une liste des principales commandes disponibles avec un préfixe personnalisé :

  • !kick [utilisateur] [raison] : Expulse un membre du serveur.
  • !ban [utilisateur] [raison] : Bannit un membre du serveur pour des infractions graves.
  • !mute [utilisateur] [raison] : Réduit les capacités d'un membre en cas de comportement perturbateur.
  • !unmute [utilisateur] : Restaure les capacités d'un membre qui a été muté.
  • !warn [utilisateur] [raison] : Avertit un membre pour une infraction aux règles du serveur.
  • !clear [nombre] : Supprime un certain nombre de messages dans un canal textuel.
  • !lockdown [canal] : Verrouille un canal pour empêcher les membres d'y envoyer des messages.
  • !unlock [canal] : Déverrouille un canal pour que les membres puissent envoyer des messages à nouveau.
  • !report [utilisateur] [raison] : Signale un utilisateur pour un comportement inapproprié.
  • !help : Affiche la liste complète des commandes disponibles.

fonctionalites anti-raid

En plus des commandes de modération, AEGIS propose diverses fonctionnalités pour faciliter la gestion de votre serveur :

  • Gestion des rôles : Gérez facilement les rôles et permissions des membres via des commandes dédiées.
  • Gestion des canaux : Créez, modifiez, et gérez les canaux de votre serveur de manière simple et rapide.
  • un systeme d'anti-raid : Configurez des filtres pour bloquer les contenus indésirables automatiquement.
  • logs : Suivez les statistiques et gardez un enregistrement des activités de modération.
